# Healing Homes Mexico

The home is the medicine. The craft is how we make it.

Architectural practice rooted in Lake Chapala. Sister product to LeadsZen, Demian, and Origen Media.

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4
- pnpm
- Supabase (shared Origen Infrastructure project, `healinghomes` schema)

## Brand

See [BRAND.md](./BRAND.md). Tokens in `lib/tokens/` are canonical. ESLint enforces no hex outside that directory.

## Development

```bash
pnpm install
pnpm dev
```

Open http://localhost:3002 (Demian on 3000, Origen Media on 3001).

## Build

```bash
pnpm build
pnpm start
```
